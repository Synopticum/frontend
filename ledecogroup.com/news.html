MSP-FET430UIF мощный инструмент для отладки и
программирования микроконтроллеров MSP430

MSP-FET430UIF -от фирмы Texas Instruments представляет собой мощный инструмент Flash эмуляции, с помощью которого можно быстро начать разработку приложений для микроконтроллеров семейства MSP430. Подключения к компьютеру происходит через USB-интерфейс, прибор можно использовать, как для отладки так и для программирования микроконтроллеров MSP430 через интерфейс JTAG или по протоколу Spy ...
